01
地基 水位由短板决定，暴力法是 O(n²) 墙
  • 任选两根线,水量 = min(两线高) × 两线间距。为什么取较矮?水从矮的那边溢走,短板决定水位——全题推理的地基。
  • 暴力:双重循环枚举每一对,O(n²),小数据能过、n 一大就 TLE。题目要 O(n),又是冲着"别双重循环"来的。
  • 例:[1,8,6,2,5,4,8,3,7] → 选下标 1(高8)和 8(高7):min(8,7)×(8-1)=49。
口诀面积 = 短板高 × 宽。抓住"短板决定水位"，后面每一步推理都从这儿长出来。
02
题眼 左右对撞：为什么每步只移动"短的"那一侧
  • left=0, right=n-1 从两头往中间夹。每轮:算当前面积、更新 max,然后只把矮的那侧往里收一格。
  • 推理链(全题唯一动脑处):往里收 → 宽度必定变小 → 想让面积有机会变大,只能寄望高度变大 → 而水位卡在短板上。
  • 所以:动短板侧 → 可能换来更高的短板(唯一"可能变好"的方向);动长板侧 → 短板没变、宽度还减 → 保证变差,纯亏。
i, j = 0, len(height) - 1
big = 0
while i < j:
    big = max(big, (j - i) * min(height[i], height[j]))
    if height[i] < height[j]:   # Python 不用给条件加括号
        i += 1               # 短板在左,收左
    else:
        j -= 1               # 短板在右(或相等),收右
return big
套路"两头对撞"双指针:每步只移动短板那一侧——因为移动长板一侧是"保证变差"。
